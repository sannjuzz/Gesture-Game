Description:
This is a real-time gesture-controlled game built using Python, OpenCV, and MediaPipe. 
Players control a paddle using hand movements captured via webcam to catch falling balls. 
The game is interactive, fun, and touch-free.

Features:
- Real-time hand detection using MediaPipe
- Control the paddle with hand gestures
- Score system: earn points for catching balls
- Game over when a ball is missed
- Lightweight and runs smoothly in VS Code

Tech Stack:
- Python
- OpenCV
- MediaPipe
- NumPy

Usage:
1. Open VS Code and navigate to the project folder.
2. Install dependencies using:
   pip install -r requirements.txt
3. Run the game:
   python gesture_game.py
4. Move your hand left and right in front of the webcam to control the paddle.
5. Catch the falling balls to score points.
6. Press 'q' to exit the game.
